Duty Manager Rotation Scheduling Tool / DM ROTA Tool

The Duty Manager or DM is a point of contact for incoming issues or "tickets" raised by the end-user. Every branch of business has a DM assigned. On a business standpoint if everything goes well, The DM will accept the ticket via the HP Service Manager software or SM9 and assign it to the right IT specialist.

There are however certain occassions where the issue / ticket isn't correctly raised to the correct DM as there are too many branch to ask on which DM should take care of the ticket and it gets confusing.

The DM ROTA Tool is a web application created to be a self-service portal wherein all the DM's for that day are recorded, At what time they will go to office? their contact number? their backup if one of them is on sick leave or on leave. This will ensure that every ticket will be taken care of, effectively reducing SLA impact and increasing productivity. Ensuring the end-user that their issues are being taken care of.

Coded using HTML, CSS, PHP using CodeIgniter framework and MySQL for the database.

Automatic Ticket Assignment Tool or ATAT

HP uses a software called the HP Service Manager software or SM9 to
collect all the issues / ticket the end-user has raised.

The Duty Manager or the DM, manually assigns these tickets to the right IT specialist.

Being the DM is overwhelming since you handle everything from assigning tickets to
informing your supervisor if there is a red alert ticket. This process is prone to human errors.

ATAT is created to ensure to lift certain responsibilities from the DM,
the machine will be the one to assign the ticket to the IT specialist using a system called the round robin assignment.

This will ensure correct ticket distribution within the team and the knowledge to be spread out.

Algorithm is coded using Python implementing the use of IMacro and OpenTwebst.
Web Application is coded using HTML, CSS, PHP using CodeIgniter framework and MYSQL for the database.

RFC Variant Comparison and Synchro

The use of this tool is SAP related.

An IT specialist daily configures jobs or processes to ensure that the variables have the correct values,
having the problem to configure multiple jobs a day this will result into waste of time and human errors.

This tool RFC Variant Comparison and Synchro ensures that the jobs are configured correctly by checking every environment in the system namely for the testing environment, quality environment and production environment.
